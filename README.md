# Assignment-2-imad
This andriod app present users with a series of Hack/Myth questions

## Features 
- Hack/Myth quiz
- Score tracking
- Answer review
- Simple UI

- ## How it works
- User answer a series of questions
- App evaluate responses
- Display score
- Highlights incorrect answers

-## Technology Used
- Java and Kotlin
- Andriod Studio
